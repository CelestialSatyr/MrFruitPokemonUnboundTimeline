Nuzlocke Rules
  1. Faint = death — If a pokemon faints, it must be permanently placed inside PC Box
  2. First encounter only — Can only catch the first pokemon encountered in each route.
  3. Nickname all Pokémon — Must nickname every pokemon caught (will be named after someone in House Fruit: Be sure to comment!)
  4. Randomized — Every wild pokemon encounter, static pokemon encounter, trainer pokemon, ability and moveset/learnset are randomized.
  5. Black Out = Game Over — Black/white out = game over.
  6. Level Cap — No pokemon can exceed level of next gym leaders highest level
  7. 4 Items per Battle — 4 Items used in-battle per battle
